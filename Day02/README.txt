## 📚 Topics Covered

### 1. Data Types

Python provides different built-in data types:

- Integer (`int`)
- Float (`float`)
- String (`str`)
- Boolean (`bool`)
- List (`list`)
- Tuple (`tuple`)
- Dictionary (`dict`)
- Set (`set`)

Example:

```python
name = "Tushar"
age = 23
salary = 25000.50
is_student = True
```

---

### 2. Strings

A string is a sequence of characters enclosed in single or double quotes.

Example:

```python
company = "KodeKloud"
```

---

### 3. String Indexing

Access characters using their position.

Example:

```python
text = "Python"

print(text[0])
print(text[-1])
```

Output:

```
P
n
```

---

### 4. String Slicing

Extract a portion of a string.

Example:

```python
text = "PythonForDevOps"

print(text[0:6])
print(text[6:])
```

---

### 5. String Handling Functions

Some commonly used string methods:

| Function | Description |
|----------|-------------|
| upper() | Converts to uppercase |
| lower() | Converts to lowercase |
| capitalize() | Capitalizes first letter |
| title() | Converts every word to title case |
| strip() | Removes leading and trailing spaces |
| replace() | Replaces text |
| split() | Splits string into a list |
| join() | Joins list elements |
| startswith() | Checks starting characters |
| endswith() | Checks ending characters |
| find() | Finds substring position |
| count() | Counts occurrences |

Example:

```python
name = "tushar"

print(name.upper())
print(name.capitalize())
```

---

### 6. String Formatting

Using **f-strings** for readable output.

Example:

```python
name = "Tushar"
role = "DevOps Engineer"

print(f"My name is {name}")
print(f"I want to become a {role}")
```

---

### 7. Escape Characters

Example:

```python
print("Hello\nWorld")
print("Python\tDevOps")
print("\"Docker\"")
```
